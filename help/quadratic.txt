[1;4mquadratic[0m

Solves ax² + bx + c = 0 and returns both roots as a tuple. Returns real roots when the discriminant is non-negative, complex conjugate roots otherwise.

[1mExamples:[0m
[0m> quadratic(1, -5, 6)
result = (3, 2)
> quadratic(1, -2, 1)
result = (1, 1)
> quadratic(1, 0, 1)
result = (i, -i)
> quadratic(1, 0, 4)
result = (2i, -2i)
