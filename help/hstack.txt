[1;4mhstack[0m

Stacks two arrays horizontally (column-wise). Ranks are promoted: a 1-D
vector is treated as a single column, and a scalar as a 1x1 block. Row
counts must match.

[1mExamples:[0m
[0m> hstack(eye(2), eye(2))
result =
⎡ 1  0  1  0 ⎤
⎣ 0  1  0  1 ⎦
> hstack((1,2), (3,4))
result =
⎡ 1  3 ⎤
⎣ 2  4 ⎦
[0m
